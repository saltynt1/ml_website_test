CIFAR10 Model Test By Nam Tran

What's this webpage about?

This is a webpage that displays the results of my Pytorch model using the CIFAR-10 builtin dataset. Since this is my first time developing a multiclass machine learning model, don't expect the results to be completely accurate. Based on the accuracy rate when training my model, this model could identify batches of images to around 80% accuracy.

Instructions

Upload an image and the webpage would tell you if the image contains an airplane, automobile, bird, cat, deer, dog, frog, horse, ship, truck, or something else. Allowed types are jpg and jpeg images.
